## Post 1: Task 3 — Heart Disease Prediction

### Post Content

🧠 **AI/ML Internship Task: Heart Disease Prediction**

I built a classification model that predicts heart disease risk using clinical data from the UCI Heart Disease dataset.

**What I did:**
• Loaded & cleaned the dataset (handled missing values, binarized target)
• Explored relationships between age, cholesterol, chest pain type, max heart rate, and heart disease risk
• Trained Logistic Regression & Decision Tree classifiers
• Evaluated with confusion matrix, ROC curve, and AUC scores

**Results:**
✅ Logistic Regression: ~85% accuracy, AUC ~0.90
✅ Decision Tree: ~82% accuracy, AUC ~0.85
✅ Top predictors: number of major vessels (ca), thalassemia type (thal), and ST depression (oldpeak)

**Key takeaway:** Logistic Regression provided the best balance of accuracy and interpretability for medical diagnosis. Feature importance analysis revealed that the number of major vessels colored by fluoroscopy is the strongest single predictor of heart disease.

## Post 2: Task 5 — Mental Health Support Chatbot

### Post Content

💙 **AI/ML Internship Task: Mental Health Support Chatbot**

I fine-tuned a language model to provide empathetic, supportive responses for stress, anxiety, and emotional wellness.

**What I did:**
• Used DistilGPT2 (82M params) as the base model — lightweight and efficient
• Fine-tuned on Facebook AI's EmpatheticDialogues dataset (25K+ conversations)
• Used Hugging Face Trainer API with custom prompt formatting
• Built both a CLI interface and a Streamlit web app

**The result:** A chatbot that responds with genuine empathy — whether someone is anxious about an interview, feeling lonely, or overwhelmed with work.

**Why this matters:** Mental health support should be accessible. While this is not a replacement for professional help, AI-powered empathetic conversations can be a valuable first step for those who need someone to talk to.

**Tech:** PyTorch, Hugging Face Transformers, DistilGPT2, Streamlit

## Post 3: Task 6 — House Price Prediction

### Post Content

🏠 **AI/ML Internship Task: House Price Prediction**

I built a regression model to predict house values using property and demographic features.

**What I did:**
• Performed feature engineering (created rooms_per_household, location score, etc.)
• Trained Linear Regression & Gradient Boosting models
• Evaluated using MAE, RMSE, and R² metrics
• Analyzed residuals and feature importance

**Results:**
📊 Gradient Boosting: R² ~0.80, MAE ~$35K
📊 Linear Regression: R² ~0.61, MAE ~$49K
📊 Top features: median income, location, avg occupancy

**Key takeaway:** Gradient Boosting significantly outperformed Linear Regression — capturing non-linear relationships in housing data. Median income alone explained most of the variance, but location-derived features added meaningful predictive power.
